# 终端服务部培训路线图

> 从零基础到全栈开发/AI 应用的完整学习路径

## 概述

终端服务部是一个核心支撑部门，负责构建和维护可靠的技术基础，同时推动知识管理和智能化工具开发与应用。
本文档作为终端服务部学习路线索引，旨在帮助新成员和现有成员系统化提升技能，涵盖从计算机基础知识到高级开发与运维实践的各个方面。

## 使用方法

先了解必要的基础知识，然后根据个人兴趣和部门需求，选择相应的专业方向进行深入学习。
在遇到问题时，可以参考相关资源或向团队成员寻求帮助。
如果有任何建议或补充内容，欢迎随时提出。
